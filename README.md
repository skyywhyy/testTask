# Тестовое задание для стажёра C++

Это проект на C++17 под Linux, состоящий из динамической библиотеки обработки данных и двух независимых программ, которые обмениваются данными через TCP-сокеты.

Необходимо реализовать три функции обработки строк, программу с двумя потоками и общим буфером, а также TCP-сервер для приёма и проверки рассчитанных значений. Программы должны корректно работать при разрыве и восстановлении соединения, а основная логика должна быть покрыта автоматическими тестами.
